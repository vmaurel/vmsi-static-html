Installation Vinted Extended Wallet

Suivez ces 4 étapes simples pour installer et configurer votre extension Vinted Extended Wallet. Le processus prend moins de 5 minutes et vous serez opérationnel immédiatement !

1

Accéder au Chrome Web Store

Rendez-vous sur la page officielle d'Vinted Extended Wallet dans le Chrome Web Store.

Cette étape garantit que vous téléchargez la version officielle et sécurisée de l'extension directement depuis Google.

Important : N'installez Vinted Extended Wallet que depuis le Chrome Web Store officiel pour votre sécurité.

Cliquez sur le bouton "Ajouter à Chrome" pour commencer l'installation.

Capture d'écran du Chrome Web Store
Page Vinted Extended Wallet avec bouton "Ajouter à Chrome"
2

Installer l'extension

Popup de confirmation Chrome
"Ajouter Vinted Extended Wallet ?"

Chrome vous demandera de confirmer l'installation. Cliquez sur "Ajouter l'extension" dans la popup qui apparaît.

L'extension se télécharge et s'installe automatiquement. Vous verrez une notification de confirmation une fois l'installation terminée.

À savoir : L'extension demande l'autorisation d'accéder aux pages Vinted pour fonctionner correctement.

L'icône Vinted Extended Wallet apparaîtra dans votre barre d'extensions Chrome.

3

Épingler l'extension

Pour faciliter l'accès à Vinted Extended Wallet, épinglez l'extension dans votre barre d'outils Chrome.

Cliquez sur l'icône des extensions (puzzle) dans Chrome, puis sur l'icône d'épingle à côté de Vinted Extended Wallet.

Recommandé : Épingler l'extension vous permet d'y accéder rapidement depuis n'importe quelle page Vinted.

L'icône Extended Wallet sera maintenant toujours visible dans votre barre d'outils.

Menu des extensions Chrome
Épinglage d'Extended Wallet
4

Activer votre licence

Interface d'activation Extended Wallet
Champ de saisie de la clé de licence

Cliquez sur l'icône Extended Wallet dans votre barre d'outils pour ouvrir le panneau de configuration.

Saisissez la clé de licence que vous avez reçue par email après votre achat.

Clé de licence : Vous l'avez reçue par email immédiatement après votre paiement. Vérifiez aussi vos spams.

Cliquez sur "Activer" et c'est terminé ! Extended Wallet est maintenant prêt à transformer votre porte-monnaie Vinted.

Félicitations !

Questions d'installation fréquentes

Je ne trouve pas l'extension dans le Web Store

Recherchez "Extended Wallet VMSI" ou utilisez le lien direct fourni dans votre email de confirmation d'achat.

J'ai perdu ma clé de licence

Contactez-nous à support@vmsi-solutions.fr avec votre email d'achat. Nous vous renverrons votre clé immédiatement.

L'extension ne s'affiche pas sur Vinted

Vérifiez que l'extension est bien activée et que vous êtes sur vinted.fr/wallet/history. Rechargez la page si nécessaire.

Extended Wallet fonctionne-t-il sur mobile ?

Extended Wallet est une extension Chrome desktop uniquement. Elle ne fonctionne pas sur l'application mobile Vinted.
